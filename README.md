# Backend Muscle Memory Platform

This repository is a **backend-first training system**.

## Structure

apps/backend   -> Express + TypeScript backend (main focus)
apps/frontend  -> Next.js training console UI
docs/          -> Learning & build checklists

## How to use

1. Start with BACKEND_TODO.md
2. Build backend level-by-level
3. Use frontend only to verify progress
4. Repeat builds for muscle memory

This repo is meant to be rebuilt many times.
